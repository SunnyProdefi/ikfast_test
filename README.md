# IKFast Test Project

This project is a collection of files and scripts for testing IKFast solutions and wrappers for various robotic configurations.

## Project Structure

- **CMakeLists.txt**: The CMake configuration file for building the project.
- **LICENSE**: The license file for the project.
- **README.md**: This file.
- **ikfast.h**: The header file containing IKFast related declarations.
- **ikfast61.cpp**: A source file generated by IKFast for UR5 configuration.
- **ikfast_wrapper.cpp**: A wrapper file for interfacing with the generated IKFast code.
- **ikfast_wrapper_arm.cpp**: A specific IKFast wrapper for arm configurations.
- **ikfast_wrapper_ec66.cpp**: A specific IKFast wrapper for ec66 configurations.
- **ikfastarm.cpp**: Additional implementations for arm-related IKFast solutions.
- **ikfastec66.cpp**: Additional implementations for ec66-related IKFast solutions.
- **main.cpp**: The main executable file for testing IKFast solutions.
- **main_arm.cpp**: Main testing file for arm-specific IKFast functionality.
- **main_arm_wrapper.cpp**: Main testing file for the arm wrapper of IKFast.
- **main_ec66.cpp**: Main testing file for ec66-specific IKFast functionality.
- **main_ec66_wrapper.cpp**: Main testing file for the ec66 wrapper of IKFast.

## Build Instructions

To build the project, ensure you have `CMake` installed, then run the following commands:

```bash
mkdir build
cd build
cmake ..
make
```

## Usage

After building the project, you can run the generated executables to test different IKFast configurations and their respective wrappers. Each main file provides specific tests and examples.
